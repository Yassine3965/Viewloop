# **App Name**: ViewLoop

## Core Features:

- User Authentication: Secure user registration and login system with redirection to YouTube after successful login.
- Dashboard: User dashboard displaying points, video submission form, and a queue of videos to watch.
- API Endpoints: API endpoints for retrieving videos, submitting points, and adding videos to the queue.
- Intelligent Video Selection: An AI tool that uses video titles provided by the ViewLoop app, attempts to intelligently match results on YouTube, then chooses an appropriate video for automated viewing. As a fallback it will occasionally open videos via a direct link.
- Automated Viewing System: Automatically watches videos based on specified duration with randomized intervals to simulate human behavior.
- Session Persistence: Ensures uninterrupted viewing sessions, handles page reloads, and manages tab closures with appropriate notifications and session termination.
- Point Management: Calculates and submits points to the server based on the video duration watched, using a tiered point system.

## Style Guidelines:

- Primary color: Strong blue (#29ABE2) evoking trust and reliability, integral to the site's automated interaction system.
- Background color: Light, desaturated blue (#E1F5FE), to create a calm and unobtrusive interface.
- Accent color: Purple (#9C27B0) to draw the user's eye, contrasting with the overall theme of blue.
- Font: 'Inter', a grotesque sans-serif, suitable for both headlines and body text due to its modern and neutral appearance.
- Consistent use of flat, modern icons to represent different features and actions.
- Clean and intuitive layout with a focus on easy navigation and clear information hierarchy.
- Subtle transitions and animations to enhance user experience without being distracting.
